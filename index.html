欢迎使用 PDF 智能点读机

这是一份专为移动端（iOS / iPadOS / Android）优化的 PDF 智能点读机 & PWA 网页应用 深度使用指南。通过本教程，你可以将该网页完美转化为一个无边框、全屏沉浸式、自带本地词态还原内核的独立文献点读 App。

📱 第一步：全屏沉浸式 App 化（PWA 模式安装）

为了获得媲美原生 App 的无缝全屏体验，强烈建议不要直接在常规浏览器标签页中使用，而是将其“添加到主屏幕”。

🍏 iPad / iPhone 用户 (Safari 浏览器)

1. 使用 Safari 浏览器 打开该点读机网页。
2. 点击浏览器右上角（iPad）或底部（iPhone）的 “分享”按钮（带有向上箭头的方框图标）。
3. 在弹出的菜单中向下滚动，找到并点击 “添加到主屏幕” (Add to Home Screen)。
4. 点击右上角确认。此时，你的主屏幕上会生成一个名为“PDF 点读机”的独立图标。
5. 退出 Safari，从主屏幕点击这个新图标打开。你会发现顶部的网址栏和底部工具栏全部消失了，转为全屏纯净模式！

🤖 Android 用户 (Chrome 或三星浏览器)

1. 使用 Chrome 浏览器（或手机自带的原生高级浏览器）打开该网页。
2. 点击右上角的 “三个点”菜单。
3. 找到并点击 “安装应用” 或 “添加到主屏幕”。
4. 确认后，系统会自动在桌面生成一个 PWA 网页应用 图标。
5. 从桌面图标点击进入，即可锁定高性能原生沙盒环境，并强制调用系统级“文档驱动”，极大提升大文件秒开速度。

📥 第二步：一键导入本地词库

本工具核心优势在于完全本地化运行，绝不上传任何隐私。首次使用需要加载你的词典数据库。

1. 获取词库：词库的网盘下载链接已内嵌在网页中，请先将扩展名为 .json 的词库文件下载到手机本地（或平板的“文件”App 中）。
2. 挂载词库：点击右上角的 ⚙️ (设置) 展开控制栏，点击 “📥 词库” 按钮。
3. 选取文件：在弹出的系统文件选择器中，选中刚刚下载的 .json 词库文件。
4. 耐心等待（关键）：由于移动端设备的内存管理机制各异，导入包含数万词条的庞大词库时，可能需要 30 秒至 1 分钟。在这期间请保持界面停留，不要切出 App。
5. 若遇闪退/卡顿：

⚠️ 注意：部分配置较低的设备在解析超大 JSON 时可能会因内存瞬间被卡住而出现闪退或无响应。不用担心，这属于正常沙盒保护。如果发生闪退，只需彻底关闭进程后重新打开 App，再次导入一次，直到屏幕弹出 “✅ 智能多模词库导入成功！” 的提示，即代表数据已永久安全地写入你设备本地的 IndexedDB 数据库中，以后打开无需重复导入。

📖 第三步：文献点读与核心功能体验

词库导入成功后，即可享受极为流畅的高清文献阅读与“哪里不会点哪里”的查词体验：

1. 📂 导入 PDF 文献

点击 “📂 PDF” 按钮，从本地选择你要阅读的 PDF 电子书或学术论文。
* 底层优化：网页内置了智能虚拟懒加载引擎。无论你的文献是几百页还是上千页，它都只会渲染你眼前看到的和上下缓冲的区域，自动释放视野外的内存，从根本上防止移动端因内存溢出（OOM）而崩溃。

2. ⚡ 触控点读与双击取词

* 标准取词：直接轻点或者双击文献中的英文单词，右侧词典面板会瞬间弹出释义，并自动调用系统级纯正美音进行发音。
* iOS 无痕点读模式：代码中专门针对苹果生态进行了底层微调。在 iPad/iPhone 上点读时，系统自带的烦人“蓝色选择双杆”和长按弹出菜单已被彻底隐形化，点击即可取词，带来丝滑、无遮挡的纯净阅读感。

3. 🧠 本地高级词态还原内核

当你点击一个衍生词（如动词过去式 stopped、复数 ies、副词 surgically、现在分词 dying 等）时，如果词库里没有这个死板的词条，内核会瞬间启动 5 级、数十种严谨的变形规则自动逆向推导，查找到其原形（如 stop / surgical / die）并呈现完美释义。

4. 🌗 🌓 模式联动（夜间护眼）

点击 “🌓 模式” 按钮，可一键切换主题：
* 白天模式：采用温润的羊皮纸护眼色（深度调和调深后的米黄书页色），消除高频亮光，让纸质感沉稳柔和。
* 反色模式（暗黑模式）：PDF 画布与词典全面进入深色调。同时，代码中加入了特制滤镜，能自动将深色模式下原本看不清的深蓝音标、蓝色词头自动转化为高亮黄加粗，确保暗光下阅读不费眼。

5. ↔️ 自由拖拽分栏

用手指按住中间的 垂直分割线 并左右滑动，可以实时调节左侧 PDF 视图与右侧词典面板的比例（完美适配横屏 iPad 与竖屏手机）。

💻 补充建议（关于电脑端）

本工具是专为移动端平板和手机量身定制的轻量化免翻墙神器。
如果你在电脑端（Windows / Mac）看论文，建议直接使用 Chrome 浏览器搭配“欧路词典”或“沙拉查词”等成熟的鼠标悬停取词插件，效率会更高、更方便！移动端的全屏沉浸点读，交给这个 PWA App 就足够了。
